Fine Tunning Hugging Face Machine Translation Model

Transformers / HuggingFace / LLM

Project

Overview

Here, I fine tunned a English to French translation pretrained model in the Hugging Face library. This process involves preparing a parallel corpus of text in the source and target languages, loading the pre-trained model and tokenizer, defining a data collator to handle the input format, and configuring training arguments. The model is then trained on the parallel corpus using a sequence-to-sequence training algorithm, gradually improving its ability to translate between the two languages.The fine-tuned model can then be used to translate new text, the performance of the resulted model is better than the original pre-trained model.
